All children should be taught to code in school Adding coding to the school curriculum means that something else must be left out. As coding is a trade it should not be included as school is reserved for teaching basic skills, such as Math or English. In the UK, a ten-fold increase in funding computer science at schools is needed, according to the Royal Society, in order to put the subject on par with physics and mathematics. The tools for writing code change too frequently for a meaningful curriculum to persist. The most popular languages in use today are Java (22 years old) and C (48 years old).

The International Olympic Committee (IOC) should create a permanent venue for the Olympic Games. Currently it has a bidding system in which different cities bid to host the games. Bid committees must pay an application fee to the IOC. The fee for hosting the 2012 Olympics was $150,000 U.S. This application fee is an irrelevant drop in the bucket.
